# Major modifications:
- added a class HDV for storing DimVar arrays with heterogeneous units
- changed the behaviour of concatenation/subsetting operators to dynamically 
  change between DimVar and HDV classes, as appropriate
- additional methods for the DimVar class
- additional properties for class u
- additional unit types (istype, typecheck)
