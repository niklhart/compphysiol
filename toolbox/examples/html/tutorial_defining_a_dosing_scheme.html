Tutorial: Defining a dosing scheme

To run a simulation in the Computational Physiology Toolbox, the individual components of a simulation task must be specified. In this tutorial, we will look at dosing.
Three routes of administration are implemented in the Computational Physiology Toolbox:
Bolus, infusion, and oral dosing. Depending on the route of administration, one of the following functions is used.
For bolus dosing, use function Bolus() in the following way. First provide compound name, we'll use Warfarin for the example, next the dosing time, 0 h , next the dose, 5 milligram, and finally the site of administration, here we take iv for an iv bolus.
dos1 = Bolus('Warfarin',0*u.h,5*u.mg,'iv')
dos1 =
Bolus dosing: Compound Time Dose Target ____________ ____ ____ ______ {'Warfarin'} 0 h 5 mg {'iv'}
Similarly, we can define an oral dosing scheme for Amitriptyline, but note that there is no need to define a site of administration here.
dos2 = Oral('Amitriptyline',0*u.day,50*u.mg)
dos2 =
Oral dosing: Compound Time Dose Formulation _________________ _____ _____ ___________ {'Amitriptyline'} 0 day 50 mg {0×0 char}
Finally, we can define an infusion by providing two time arguments, start and duration of the infusion, rather than just a single dosing time. Otherwise the syntax is like for the bolus function.
dos3 = Infusion('Warfarin',0*u.h,5*u.mg,1*u.h,'iv')
dos3 =
Infusion dosing: Compound Tstart Tstop Dose Rate Duration Target ____________ ______ _____ ____ _______________ ________ ______ {'Warfarin'} 0 h 1 h 5 mg 1.3889e-09 kg/s 1 h {'iv'}
There is also another ways to specify an infusion by its rate rather than duration, which is explained in the function help.
Make sure to use the correct unit types! Time-type arguments, such as dosing time and infusion duration, must have time-type units. The dose can be given in different units, but if the dosing scheme shall be used in a simulation task, the dosing unit must match the internal dose unit of the model used.
To specify multiple dosing, simply provide dosing time as a vector rather than a scalar.
Here we specify a once-daily oral administration of Amitriptyline for one week
Oral('Amitriptyline',(1:1:7)*u.day,50*u.mg)
ans =
Oral dosing: Compound Time Dose Formulation _________________ _____ _____ ___________ {'Amitriptyline'} 1 day 50 mg {0×0 char} {'Amitriptyline'} 2 day 50 mg {0×0 char} {'Amitriptyline'} 3 day 50 mg {0×0 char} {'Amitriptyline'} 4 day 50 mg {0×0 char} {'Amitriptyline'} 5 day 50 mg {0×0 char} {'Amitriptyline'} 6 day 50 mg {0×0 char} {'Amitriptyline'} 7 day 50 mg {0×0 char}
If the doses change between administration events, the dose itself can also be provided as a vector.
In any of these cases, we end up with an object of class Dosing (more precisely, a subclass thereof). This is the internal toolbox format for storing dosing schemes. Fur such objects, further functions are defined.
First, a plus operator allows to combine two dosing schemes. For example, we can combine an Amitriptyline loading dose and a once daily maintainance dose
Oral('Amitriptyline',0*u.day,100*u.mg) + Oral('Amitriptyline',(1:7)*u.day,50*u.mg)
ans =
Oral dosing: Compound Time Dose Formulation _________________ _____ ______ ___________ {'Amitriptyline'} 0 day 100 mg {0×0 char} {'Amitriptyline'} 1 day 50 mg {0×0 char} {'Amitriptyline'} 2 day 50 mg {0×0 char} {'Amitriptyline'} 3 day 50 mg {0×0 char} {'Amitriptyline'} 4 day 50 mg {0×0 char} {'Amitriptyline'} 5 day 50 mg {0×0 char} {'Amitriptyline'} 6 day 50 mg {0×0 char} {'Amitriptyline'} 7 day 50 mg {0×0 char}
Second, dosing objects can be queried for their compounds using function compounds():
compounds(dos1)
ans = 1×1 cell array
{'Warfarin'}
compounds(dos1+dos2)
ans = 1×2 cell
'Amitriptyline''Warfarin'

Exercise

Here's a small exercise for getting used to defining dosing schemes.
The Paul-Ehrlich-Society recommends (amongst others) the following dosing scheme for treatment of a moderate community-acquired pneumonia:
  • Initial treatment (first three days): thrice daily 2.2 gram Amoxicillin 30 minute i.v. infusion and twice daily 500 milligram Clarithromycin per oral
  • Subsequent treatment: thrice daily 1000 milligram Amoxicillin per oral
Implement this scheme as a Dosing object for the first week of treatment. Try to do the computation in the toolbox; you can check your result in the solutions.
